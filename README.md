# site-profile

## Instalasi

Jalankan perintah di bawah di folder aplikasi:

```
mim app install site-profile
```

## Event Listener

1. `profile:created` `(object $page)`
1. `profile:deleted` `(object $page)`
1. `profile:updated` `(object $page)`

## EndPoints

### `GET /profile/(:name)`

### `GET /profile/feed.xml`
